# 🎉 Fun Calculator 🧮

Welcome to the **Fun Calculator!** This is a beginner-friendly Python script that lets users **add, subtract, multiply, and divide** two numbers — with a dash of personality and fun emojis. 😎✨

---

## 📌 Features

- Accepts **decimal inputs** using `float()`
- Performs basic arithmetic operations:
  - ➕ Addition
  - ➖ Subtraction
  - ✖️ Multiplication
  - ➗ Division
- Prints out all results in a readable format
- Beginner-friendly with well-commented code
- Designed to be fun and educational 🎓🎈

---

## 🚀 How to Run

1. Make sure you have **Python 3** installed on your machine.
2. Download or copy the script into a file named `fun_calculator.py`.
3. Open a terminal or command prompt.
4. Run the script:

```bash
python fun_calculator.py
